## **5. Player Control (10pts)**

**Player Controls:**
The player controls a car using WASD or arrow keys.

* **W / Up Arrow** moves forward
* **S / Down Arrow** slows down or reverses
* **A / D** steers left and right

The player’s goal is to avoid police cars and survive as long as possible.

---

## **6. Basic Gameplay (20pts)**

### **Goals/Objectives:**

Survive for 3 minutes while avoiding police cars.
Collect powerups.
Get the highest score possible.

### **Game Explanation:**

You spawn on a City map and immediately begin driving. Police units chase the player. As the car drives, the score increases based on coins grabed. Police cars try to collide with the player to end the game.

### **How the Game Ends:**

* The timer reaches 0 = Win
  **OR**
* The player’s car collides with a police car and “dies” =Lose

After the game ends, a Game Over screen appears showing the final score with options to Retry or go back to the Main Menu.
